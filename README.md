# Auto Scheduling System

A full-stack scheduling platform designed to manage tasks and integrate third-party services.  
Built with FastAPI (Backend) and Nuxt 3 (Frontend), following a structured and modular architecture.

---

## Architecture

├── Backend/ # FastAPI + MongoDB

└── Frontend/ # Nuxt 3 + TypeScript

### Backend
- RESTful API design
- MongoDB integration
- Layered architecture (Router / Service / CRUD)
- Docker support
- Unit testing with Pytest

### Frontend
- Nuxt 3 + Vue 3 (Composition API)
- TypeScript
- Component-based UI architecture
- Unit testing (Vitest)
- E2E testing (Playwright)

---

## Features

- Task management
- Account linking (GitHub, Jira, Moodle)
- OAuth integration
- RESTful API
- Health check endpoint
- Unit and E2E testing support
